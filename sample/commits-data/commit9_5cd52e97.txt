cuj: Google meet CUJ test case

A remote test calling Google meet service to do the google meet testing.
Different sizes of conference room are designed as different sub tests.

To run Google Meet CUJ, an external static room server has to be set up
and the test will conntect to join static meeting rooms.

BUG=None
TEST=tast run -var=ui.cuj_username=x -var=ui.cuj_password=x -var=ui.meet_account=x -var=ui.meet_url=x $DUT ui.GoogleMeetCUJ.*
